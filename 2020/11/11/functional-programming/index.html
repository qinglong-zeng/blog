# 柯里化

柯里化函数

  1. 使用递归.
  2. 如果提供的参数（args）数量足够，请调用传递的函数fn.
  3. 否则，使用Function.prototype.bind（）返回需要其余参数的函数fn .
  4. 如果要柯里化函数接受可变数量参数的函数（可变函数，例如Math.min（）），则可以选择将参数数量传递给第二个参数arity .
const curry = (fn, arity = fn.length, ...args) =>
  arity <= args.length ? fn(...args) : curry.bind(null, fn, arity, ...args);

curry(Math.pow)(2)(10); // 1024
curry(Math.min, 3)(10)(50)(2); // 2

# pipe && compose

# pipe

将Array.prototype.reduce（）与散布运算符（...）结合使用以执行从左到右的函数合成.

  1. 将Array.prototype.reduce（）与散布运算符（...）结合使用以执行从左到右的函数合成.
  2. 对异步功能执行从左到右的功能组合。.
const pipeFunctions = (...fns) =>
  fns.reduce((f, g) => (...args) => g(f(...args)));

const add5 = x => x + 5;
const multiply = (x, y) => x * y;
const multiplyAndAdd5 = pipeFunctions(multiply, add5);
multiplyAndAdd5(5, 2); // 15

# pipeAsyncFunctions

对异步功能执行从左到右的功能组合。

  1. 使用Array.prototype.reduce（）和散布运算符（...）使用Promise.prototype.then（）进行功能组合 .
  2. 这些函数可以返回正常值，承诺或异步的组合，并通过等待返回.
  3. 所有函数必须接受一个参数.
const pipeAsyncFunctions = (...fns) =>
  arg => fns.reduce((p, f) => p.then(f), Promise.resolve(arg));

const sum = pipeAsyncFunctions(
  x => x + 1,
  x => new Promise(resolve => setTimeout(() => resolve(x + 2), 1000)),
  x => x + 3,
  async x => (await x) + 4
);
(async() => {
  console.log(await sum(5)); // 15 (after one second)
})();

# compose

执行从右到左的功能组合

  1. 使用Array.prototype.reduce（）执行从右到左的函数合成.
  2. 最后一个（最右边）函数可以接受一个或多个参数。其余功能必须是一元的.
const compose = (...fns) =>
  fns.reduce((f, g) => (...args) => f(g(...args)));

const add5 = x => x + 5;
const multiply = (x, y) => x * y;
const multiplyAndAdd5 = compose(
  add5,
  multiply
);
multiplyAndAdd5(5, 2); // 15

# composeRight

执行从左到右的功能合成。

  1. 使用Array.prototype.reduce（）执行从左到右的函数合成。
  2. 第一个（最左侧）函数可以接受一个或多个参数。其余功能必须是一元的。
const composeRight = (...fns) =>
  fns.reduce((f, g) => (...args) => g(f(...args)));

const add = (x, y) => x + y;
const square = x => x * x;
const test = x => x * 2
const addAndSquare = composeRight(add, square, test);

(...args) => test((...args) => square(add(...args))(..args))

test(square(add(1, 2)))

addAndSquare(1, 2); // 18

# debounced

创建一个防反跳函数，该函数ms会将调用提供的函数延迟到自上次调用该函数以来至少经过了毫秒。

  • 每次调用去抖动功能时，请使用清除当前的挂起超时，clearTimeout()并用于setTimeout()创建一个新的超时，该超时将延迟调用该功能，直到至少ms经过毫秒。
  • 使用Function.prototype.apply()该应用this上下文的功能，并提供必要的参数。
  • 忽略第二个参数，ms将超时设置为默认值0ms。
const debounce = (fn, ms = 0) => {
  let timeoutId;
  return function(...args) {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => fn.apply(this, args), ms);
  };
};

window.addEventListener(
  'resize',
  debounce(() => {
    console.log(window.innerWidth);
    console.log(window.innerHeight);
  }, 250)
); // Will log the window dimensions at most every 250ms

# throttle

创建一个限制的函数，每等待一个毫秒最多只能调用一次提供的函数。

  • 使用setTimeout()和clearTimeout()限制给定的方法fn。
  • 使用Function.prototype.apply()该应用this上下文的功能，并提供必要的arguments。
  • 用于Date.now()跟踪上一次调用限制功能的时间。
  • 使用变量inThrottle来防止第一次执行fn和下一个循环之间的竞争状态。
  • 省略第二个参数，wait以将超时设置为默认值0 ms。
const throttle = (fn, wait) => {
  let inThrottle, lastFn, lastTime;
  return function() {
    const context = this,
      args = arguments;
    if (!inThrottle) {
      fn.apply(context, args);
      lastTime = Date.now();
      inThrottle = true;
    } else {
      clearTimeout(lastFn);
      lastFn = setTimeout(function() {
        if (Date.now() - lastTime >= wait) {
          fn.apply(context, args);
          lastTime = Date.now();
        }
      }, Math.max(wait - (Date.now() - lastTime), 0));
    }
  };
};

window.addEventListener(
  'resize',
  throttle(function(evt) {
    console.log(window.innerWidth);
    console.log(window.innerHeight);
  }, 250)
); // Will log the window dimensions at most every 250ms
